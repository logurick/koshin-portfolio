# 向心クラブ ホームページ

延岡市で活動する軟式野球チーム「向心クラブ」の公式ホームページです。

## 今回のデザイン方針

- スポーツチームらしい大きなタイポグラフィ、濃いグリーン、アクセントカラーを採用
- NEWS / GAME / TEAM / SCHEDULE / CLUB / RECRUIT / PARTNER / CONTACT のセクション構成
- 画像が少ない状態でも見栄えが崩れにくい、CSSベースのヒーロービジュアル
- スマートフォンでも横スクロールしにくいレスポンシブ対応

## ページ構成

| セクション | 内容 |
| --- | --- |
| NEWS | `articles/` の記事一覧 |
| GAME | 次回試合、試合結果、練習試合募集の導線 |
| TEAM | チーム紹介、活動方針 |
| SCHEDULE | 練習・試合・募集の予定エリア |
| CLUB | チーム概要 |
| RECRUIT | メンバー募集 |
| PARTNER | 応援・協力先の掲載エリア |
| CONTACT | 問い合わせ情報 |

## 記事投稿

`articles/` フォルダに `.txt` ファイルを1記事1ファイルで追加します。

```txt
---
title: 記事タイトル
date: 2026-06-16
category: お知らせ
excerpt: 一覧に表示したい短い説明文。省略可。
---
本文をここに書きます。

## 見出し

- 箇条書きも簡易対応しています
- 詳細ページに反映されます
```

対応している簡易記法は次の通りです。

- `## 見出し`
- `### 小見出し`
- `- 箇条書き`
- `**強調**`

## ローカル確認

```bash
npm run build
```

生成された `dist/index.html` をブラウザで開くと確認できます。

開発中に自動ビルドしたい場合は次を使います。

```bash
npm run dev
```

## GitHub Pages

`.github/workflows/pages.yml` が、`main` ブランチへの push 時にサイトをビルドして Pages へデプロイします。

リポジトリ設定の Pages は、Source を `GitHub Actions` にしてください。

## 変更しやすい箇所

チーム情報、試合情報、活動予定などの固定文言は `scripts/build-site.js` の冒頭にまとめています。

デザイン調整は `assets/site.css` を編集してください。
