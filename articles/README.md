# 記事の追加方法

`articles/` に `.txt` ファイルを1記事1ファイルで置くと、GitHub Pages公開時に自動でホームページへ反映されます。

## ファイル例

```txt
---
title: 練習試合のお知らせ
date: 2026-07-01
category: 試合予定
---
7月の練習試合についてお知らせします。

日時や場所が決まり次第、追記します。
```

## ルール

- 文字コードはUTF-8にしてください。
- ファイル名は半角英数字・ハイフン中心がおすすめです。
- `title` は記事タイトルです。
- `date` は `YYYY-MM-DD` 形式で書いてください。
- `category` は未指定でも構いません。
- `---` より下が本文です。空行を入れると段落になります。
